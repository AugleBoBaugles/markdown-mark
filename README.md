# Markdown Mark

Markdown Mark is an AI-powered tool that converts messy notes, URLs, or raw text into clean, structured Markdown. Built for students, developers, and anyone who works with documentation, Markdown Mark helps you turn chaos into clarity with a single click.


## ✨ Features

- **Automatic Markdown Formatting**  
  Cleans up text, organizes content, and outputs polished Markdown.

- **URL + Text Support**  
  Paste raw notes or a link — Markdown Mark handles both.

- **Quick-Action Buttons**  
  Refine, shorten, expand, or reformat your output instantly.

## Set Up
### Update ENV
1. Duplicate the *template.env* and rename to be *.env*
2. Add your OPENAI API Key to the .env file

### Install Dependencies
```bash
npm i
```

```bash
cd frontend
npm i
```

## Run the App
From the root run this
```bash
node server.js
```
Open a new terminal and run this
```bash
cd frontend
npm run dev
```



Your up and running! Go to [http://localhost:5173](http://localhost:5173)
